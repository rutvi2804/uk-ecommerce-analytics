# UK E-Commerce Analytics

[![Live Dashboard](https://img.shields.io/badge/Live%20Dashboard-Visit%20Now-blue?style=for-the-badge)](https://uk-ecommerce-analytics.onrender.com)

End-to-end data analytics project using SQL Server, dbt, 
Python, and Plotly Dash.

## 🚀 Live Dashboard
👉 [https://uk-ecommerce-analytics.onrender.com](https://uk-ecommerce-analytics.onrender.com)

## Dataset
UK E-Commerce transactions — 541,909 rows from Kaggle

## Tech Stack
- **SQL Server Express** — database
- **dbt Core** — data transformation and modeling
- **Python + pandas** — analysis and EDA
- **Plotly Dash** — interactive dashboard
- **Render.com** — cloud deployment

## Key Features
- RFM Customer Segmentation — 8 segments across 4,334 customers
- Cohort Retention Analysis — 13 monthly cohorts
- Sales Trend Analysis — monthly revenue and orders
- Regional Performance — Europe choropleth map
- Product Analysis — top 15 products by revenue

## Project Structure
- `data/` — raw source data and CSV exports
- `dbt_project/` — dbt models (staging, intermediate, marts)
- `notebooks/` — EDA analysis
- `dashboard/` — Plotly Dash application
