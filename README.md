# UK E-Commerce Analytics

End-to-end data analytics project using SQL Server, dbt, Python and Plotly Dash

## Dataset
UK E-Commercetransactions from Kaggle - 500K + rows of retail transcation data

## Tech Stack
- SQL Server Express — database
- dbt Core — data transformation and modeling
- Python + pandas — analysis
- Plotly Dash — interactive dashboard
- Render.com — deployment

## Project Structure
- data/raw - raw source dara
- dbt_project/ - Dbt models 
- notebooks/ - EDA analysis
- dashboard/ - Plotly Dash app
